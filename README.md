## ✨ Key Features

- **🛍️ Digital Asset Marketplace:** Browse and purchase game assets with an intuitive UI.
- **🛡️ Secure Escrow System:** Built-in consignment model to protect buyer funds until digital delivery is confirmed.
- **🔐 Robust Authentication:** Secure Login and Registration system with session management.
- **🛒 Smart Cart System:** Dynamic shopping cart that tracks items seamlessly.
- **📱 Responsive UI:** Fully responsive design built with Tailwind CSS, ensuring a smooth experience across desktop and mobile.
- **⚡ Serverless Ready:** Optimized for Vercel deployment with secure PostgreSQL pooling.
